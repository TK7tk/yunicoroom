# Extraction 01: Autonomous Agents Trends 2025

## Source Type
Web Search: "autonomous agents trends 2025"

## Key Trends
1.  **Shift from GenAI to Agentic AI**: Moving from "chatbots" that wait for prompts to "agents" that proactively act, plan, and execute.
2.  **Agent Archetypes**:
    -   **Agentic RAG**: Combining retrieval with reasoning.
    -   **Voice Agents**: High fluency, human-like interaction.
    -   **Computer Using Agents (CUA)**: Operating GUIs like humans.
    -   **Coding Agents**: Autonomous devops and coding.
3.  **Multi-Agent Systems**: Agents collaborating via protocols like **MCP (Model Context Protocol)**.
4.  **Enterprise Adoption**: Moving from pilot to production. 88% of early adopters see positive ROI.
5.  **governance & Security**: Focus on "Responsible AI" and security as agents get more power.

## Technical Breakthroughs
-   **Planning & Reasoning**: Breaking down complex goals into steps.
-   **Tool Use**: Ability to use external APIs, browsers, and software.
-   **Memory**: Long-term context retention.
