# Source 5: Security & Enterprise Features

## Security Suite (GitHub Advanced Security)
- **Dependabot**: Auto-updates vulnerable libraries.
- **CodeQL**: Deep semantic code analysis for vulnerabilities.
- **Secret Scanning**: Prevents committing API keys/passwords.

## Enterprise Administration
- **SAML/SSO**: Integration with corporate identity (Azure AD, Okta).
- **Audit Logs**: Compliance and activity tracking.
- **Managed Users**: Centralized user lifecycle management.
- **Policies**: Enforce 2FA, branch protection, and allowed actions organization-wide.
